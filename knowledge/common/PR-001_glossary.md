## 通用術語

| 術語 | 原文 | 說明 |
|------|------|------|
| 韌體 | Firmware | 嵌入硬體的軟體 |
| 燒錄 | Flash/Program | 將韌體寫入記憶體 |
| 驅動程式 | Driver | 作業系統與硬體間的橋樑 |
| 軟體 | Software | 一般應用程式 |
| 硬體 | Hardware | 實體裝置 |

## Debug 相關

| 術語 | 原文 | 說明 |
|------|------|------|
| 斷點 | Breakpoint | 程式執行中斷點 |
| 監看式 | Watchpoint | 記憶體/暫存器監看 |
| 單步執行 | Single Step | 一次執行一行 |
| 軟體斷點 | Software Breakpoint | 0xBE 指令替換 |
| 硬體斷點 | Hardware Breakpoint | DWT 暫存器 |

## USB 相關

| 術語 | 原文 | 說明 |
|------|------|------|
| MSC | Mass Storage Class | USB 大容量儲存類別 |
| HID | Human Interface Device | 人機介面裝置 |
| DFU | Device Firmware Update | 韌體更新 |
| URB | USB Request Block | USB 請求區塊 |
| EP | Endpoint | USB 端點 |

| Push | Push | 推送到遠端 |
| Pull | Pull | 從遠端拉取 |
