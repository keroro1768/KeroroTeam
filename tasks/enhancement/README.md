# Enhancement Backlog — 強化待辦清單

> 位置：`D:\AiWorkSpace\KeroroTeam\tasks\enhancement\
> 所有成員都可以提案

---

## 提案格式

建立新資料夾：`E001_xxx/`

```
E001_xxx/
├── Task.md       ← 強化構想
├── PLAN.md      ← 執行計畫
├── WORKLOG.md   ← 工作日誌
└── VERIFY.md   ← 完成後填寫
```

---

## 提案樣板（E001_xxx/Task.md）

```markdown
# Task.md — E001

## 基本資訊

| 欄位 | 內容 |
|------|------|
| ID | E001 |
| 標題 | {強化構想名稱} |
| 提案人 | {你的代號} |
| 提案日期 | YYYY-MM-DD |
| 狀態 | ⏳ 待審 |

## 當前狀態

{描述現在的問題或可以更好的地方}

## 預期效益

{做完後的改善}

## 預估時間

{X 小時 / X 天}

## 標籤

#enhancement #(?)

## 審核

| 項目 | 結果 |
|------|------|
| Dororo 審核 | 待填 |
| 價值確認 | 待填 |
| 負責人 | 待認領 |
```

## 提案流程

1. 想到 Enhancement → 建立 `E0XX/` 資料夾
2. 填寫 `Task.md`
3. Dororo 審核價值
4. 確認後安排負責人
5. 執行，完成後填 `VERIFY.md`
